机器学习基础概念

机器学习是**人工智能**的一个重要分支，让计算机能够从数据中学习并做出预测或决策。

主要类型

  • 监督学习：使用标记数据训练模型
    • 分类：预测类别（如邮件是否为垃圾邮件）
    • 回归：预测数值（如房价预测）
  • 无监督学习：从无标记数据中发现模式
    • 聚类：将相似数据分组
    • 降维：简化数据复杂度
  • 强化学习：通过试错学习最优策略

常用算法

# 简单的线性回归示例
from sklearn.linear_model import LinearRegression
import numpy as np

# 创建数据
X = np.array([[1], [2], [3], [4], [5]])
y = np.array([2, 4, 6, 8, 10])

# 训练模型
model = LinearRegression()
model.fit(X, y)

# 预测
prediction = model.predict([[6]])
print(f"预测结果: {prediction[0]}")

机器学习的核心是找到数据中的**模式**和**规律**，然后用这些规律对新数据进行预测。
